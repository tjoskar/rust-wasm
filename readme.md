# Rust + WebAssembly = ❤️

### Setup

Follow this guide: https://rust-lang-nursery.github.io/rust-wasm/setup.html

### Terminal

```
$ cd rust-bin
$ cargo build [--release]
$ cargo run // or ./target/release/hello_world
```

### Web

```
$ cd web-hello-world
$ make
$ python3 server.py
```

Visit http://localhost:9000 and you are good to go
